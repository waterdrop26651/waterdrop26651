### 🔬 Research Experience

**MM ALIGNER: Safeguarding Multimodal Large Language Models through Representation Calibration**  
*ACM CCS '26 (CCF-A, Under Review)* | **Co-Author (Student 2nd Author)**

- **Background**: Addressed security failures in Multimodal LLMs by revealing "Representation Drift" as the root cause of jailbreaks from a representation learning perspective.
- **Methodology**: Proposed the `MMALIGNER` defense framework based on representation calibration, utilizing hard lower-bound constraints and soft upper-bound regularization. Achieved 99% malicious input rejection rate across models like LLaVA and Qwen with <2% general capability loss.
- **My Work**: Built the geometric analysis evaluation pipeline based on internal representation spaces. Developed an automated evaluation framework, conducted SFT/DPO fine-tuning, and reproduced 9 mainstream Vision-Language security defense baselines (e.g., AdaShield, ECSO). Led comprehensive testing across security and general capability benchmarks.
